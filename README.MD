
##去除openwrite验证码插件

    比较讨厌openwirte的公众号引流功能，写了个这个谷歌插件就是为了不关注公众号。
    提供了2种方式：
    1. 直接js去除网页openwrite微信引流_自动展开全文----有概率页面展示不全
    2. 代码获取bolgId然后直接ajax请求验证码
    
## 使用说明
    下载项目，然后浏览器加载已解压的插件方式
